# ELASMOCITESINDEX
A R shiny app showing detailes  of CITES listed sharks and rays
ELASMOCITESINDEX is an interactive R Shiny web application designed to summarize and visualize information about CITES-listed elasmobranch species (sharks and rays).
The app integrates data on IUCN Red List status, CITES appendices, trade-related information, and geographical distribution to provide a single access point for researchers, managers, and students working on shark conservation.

**Features**

 Species selection: Choose any CITES-listed shark species

 Conservation status: View IUCN Red List status
 
 Brief description of the species

CITES appendix info: Shows trade regulation category and listing year

Trade insights: Highlights key trade notes and observations from official databases

Geographical range: Displays global distribution and countries of occurrence

** **Data Disclaime**r**

The dataset provided in this repository contains  publicly available data compiled from official sources such as:

IUCN Red List (assessment.csv in this repositiory)  

CITES Trade Database ( CITES_info.csv)   

transformed and combined database used for the application (citesdb.csv)
